# Tagebook

<img width="1512" height="864" alt="image" src="https://github.com/user-attachments/assets/05e81c61-61e1-4210-a26c-fa6726068dcf" />

A language learning journal app. Write in your target language and get inline grammar analysis, translations, and word breakdowns powered by a local LLM (Ollama). Save phrases for review and track your progress over time.

Built with React, TypeScript, Tiptap, and Vite.
